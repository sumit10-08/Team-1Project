HTML TAGS

1. b Tag

This content is bold

2. i Tag

This content is italic

3. em Tag

This content is italic [em:emphasis]

4. strong Tag

This content is bold (which text is important)

5. u Tag

This content is underlined

6. mark Tag

It is used to highlight a text or content.

7. Strike Tag

It is used to draw a thin line on the text

8. sup Tag(Superscript)

It is used to displayed the content lightly above to the normal text.

x2+y2+2xy=(x+y)2

9. sub Tag(Subscript)

It is used to displayed the content lightly below to the normal text.

H20
CO2

10. del Tag(Deleted)

Delected tag is used to displayed the deleted content.

11. ins Tag(Inserted)

Inserted tag is used to displayed the content which is added.

12. big Tag

It is used to increase the font size of the text.

13. small Tag

It is used to decrease the font size of the text.
