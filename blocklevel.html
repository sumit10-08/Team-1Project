BLOCK LEVEL ELEMENT

A Block level element always start with a new line and takes the full width of web page, form left to right. The P and the DIV elements are both block elements, and they will always start on a new line and take up the full width available (stretches out to the left and right as far as it can).


for example:
OSI REFERENCE MODEL
APPLICATION LAYER
PRESENTATION LAYER
SESSION LAYER
TRANSPORT LAYER
NETWORK LAYER
DATA-LINK LAYER
PHYSICAL LAYER

INLINE ELEMENT

Inline element does not start with new line and take width as per the requirement.The SPAN element is an inline element, and will not start on a new line and only takes up as much width as necessary.


for example: TCP/IP IMPLEMENTED MOFDEL APPLICATION LAYER TRANSPORT LAYER INTERNET LAYER NETWORK ACCESS LAYER
