HTML FORMATTING

HTML also defines special elements for defining text with a special meaning.
The formatting tags are divide into two categories:
1. Physical tags, used for text styling(visula appearance of the text)
2. Logical or Semantic tags are usd to add semantic value to the parts of the text.


HTML Tags & Attributes

  • (b)- Bold text
  • (strong)- Important text
  • (i)- Italic text
  • (em)- Emphasized text
  • (mark)- Marked text
  • (small)- small text
  • (del)- Deleted text
  • (ins)- Inserted text
  • (sub)- Subscript
  • (sup)- Superscript
